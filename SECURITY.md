# Security

This repository is not production-ready and currently contains only a mock generation
backend.

Do not publish secrets, model access tokens, private datasets, generated credentials,
or proprietary audio assets in issues, pull requests, benchmark folders, or generated
artifacts.

If this project becomes a public GitHub repository, use GitHub private security
advisories for vulnerability reports when available. If private advisories are not
enabled, contact the repository maintainers through a non-public channel before
posting exploit details publicly.
